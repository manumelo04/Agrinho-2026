# 🌱 Agro Forte, Futuro Sustentável

Um site institucional sobre o agronegócio moderno, destacando sustentabilidade, tecnologia e produção responsável no campo.

## 📌 Sobre o projeto

O **Agro Forte** é uma landing page que apresenta como o agronegócio pode unir produtividade e preservação ambiental por meio de inovação tecnológica, gestão inteligente de recursos e energias renováveis.

## 🚀 Funcionalidades

- Navegação suave entre seções
- Layout responsivo e moderno
- Seção de sustentabilidade com cards informativos
- Estatísticas de impacto e eficiência
- Depoimentos de produtores e especialistas
- Lista de parceiros do setor
- Botão “voltar ao topo”
- Design focado em experiência do usuário

## 🧰 Tecnologias utilizadas

- HTML5
- CSS3 (arquivo `style.css`)
- JavaScript (interações simples e scroll suave)
- Google Fonts (Poppins)

## 📁 Estrutura do projeto
Acesse a pasta do projeto:
cd agro-forte
Abra o arquivo index.html no navegador.
🎯 Objetivo

Promover a conscientização sobre práticas sustentáveis no agronegócio, mostrando como tecnologia e responsabilidade ambiental podem caminhar juntas.
